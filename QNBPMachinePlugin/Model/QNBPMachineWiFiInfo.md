# QNBPMachineWiFiInfo

WiFi配置信息，用途见[配网](../../FAQ.md#配网)

#### 成员变量

| 名称        | 类型     | 描述                         |
|-----------|--------|----------------------------|
| ssid      | string | WiFi名称                     |
| pwd       | string | WiFi密码                     |
| serverUrl | string | 服务器地址, 用于血压计WiFi通道上传测量数据使用 |
| otaUrl    | string | OTA升级地址                    |
