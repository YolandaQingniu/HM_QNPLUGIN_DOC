QNScaleUser

QNScaleUser

秤端用户信息，继承QNUser

成员变量

名称 类型 描述
index number 秤端用户序号, index == 0时, 执行注册秤端用户操作
当index 为 1~8 时，执行注册秤端用户操作
当visitorMode为true时，此设置无效
key number 范围0~65535，默认1000
当visitorMode为true时，此设置无效
visitorMode boolean 为true时，使用游客模式进行连接
为false时，根据上述的index和key进行用户注册或访问

buildQNScaleUser

构建一个QNScaleUser实例对象

参数
名称 类型 描述
userId string | undefined 用户Id
gender QNGender 性别
age number 年龄：3 - 80岁，整形
height number 身高：40.0 - 240.0 cm
sportMode boolean 是否是运动员
index number 秤端用户序号, index == 0时, SDK内置执行注册秤端用户操作
key number 秤端用户秘钥，范围0~65535
visitorMode boolean 是否是访客连接设备, 当设置true时本次连接不会向秤请求注册秤端用户,且访客用户信息仅当次连接测量有效,断开连接后访客信息清除
callback QnResultCallback 回调方法，返回构造的状态码 number 详情见状态码列表
返回值
类型 描述
QNScaleUser 秤端用户
