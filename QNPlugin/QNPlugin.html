QNPlugin

SDK初始入口组件

主要功能：

  • 蓝牙设备基本管理
  • SDK授权信息验证
  • 系统蓝牙状态监听、扫描、连接

API

getInstance

获取一个QNPlugin的单例对象

参数
名称 类型 说明
context common.ApplicationContext 获取单例需传入应用级别的的上下文
返回值

类型：QNPlugin

initSdk

根据 appId进行SDK初始化，在初始化过程中会根据所使用的 appId与 配置文件进行可使用权限验证。

建议在 onWindowStageCreate后进行初始化。

demo中所使用的appid（123456789）仅作为配合demo及调试使用，测试id具有不稳定性与随机变化性。接入方在正式使用时必须使用正式的 appId

参数
名称 类型 说明
appId string Yolanda提供的唯一标识
fileName string 配置文件的文件名，必须放在项目工程的AppScope/resources/rawfile目录下
callback QNResultCallback 回调方法，返回初始化的结果详情见状态码列表

setLogListener

设置日志输出监听，用于APP记录SDK输出的日志信息。

参数
名称 类型 说明
value QNLogListener | undefined 设置日志输出监听

setScanListener

设置扫描状态监听器，主要用于监听蓝牙启动或停止扫描的动作监听。

参数
名称 类型 说明
value QNScanListener | undefined 设置扫描状态监听

getSysBleStatus

获取当前系统蓝牙状态

返回值

类型：QNSysBleStatus

startScan

启动蓝牙扫描

返回值

类型：number 详情见状态码列表

stopScan

停止蓝牙扫描
