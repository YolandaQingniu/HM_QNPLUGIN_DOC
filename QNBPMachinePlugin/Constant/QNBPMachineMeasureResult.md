# QNBPMachineMeasureResult

测量结果状态

| 名称      | 含义        |
|---------|-----------|
| UNDONE  | 未测量完成     |
| ERR1    | 设备未标定     |
| ERR2    | 袖带脱离/动作过大 |
| ERR3    | 高压检测失败    |
| ERR4    | 低压检测失败    |
| ERR5    | 未检测到血压脉动  |
| ERR6    | 测量超时      |
| ERR7    | 过度加压      |
| ERR8    | 加压过程中出现漏气 |
| ERR9    | 未连接袖带     |
| SUCCESS | 测量成功      |
