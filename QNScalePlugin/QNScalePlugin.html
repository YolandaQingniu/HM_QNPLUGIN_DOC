QNScalePlugin

QNScalePlugin

家用秤设备管理

成员变量

名称 类型 描述
dataListener QNScaleDataListener | undefined 数据监听器
wifiListener QNScaleWiFiListener | undefined 配网监听器
scanListener QNScaleScanListener | undefined 扫描监听器
statusListener QNScaleStatusListener| undefined 状态监听器
eventListener QNScaleEventListener| undefined 事件监听器
userEventListener QNUserScaleEventListener| undefined 设备用户管理监听器

API

initPlugin

初始化 QNScalePlugin

参数
名称 类型 说明
plugin QNPlugin 基础管理类
返回值

类型：number 详情见状态码列表

connectDevice

连接设备

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
operate QNScaleOperate 设置项
返回值

类型：number 详情见状态码列表

cancelConnectDevice

断开设备连接

参数
名称 类型 说明
device QNScaleDevice 家用秤设备

getWeightLb

单位转换（kg ==> lb）

参数
名称 类型 说明
weight string 测量数据返回的weight字符串
返回值
类型 说明
string 对应lb单位数值的字符串,一位有效数字

getWeightJin

单位转换（kg ==> 斤）

参数
名称 类型 说明
weight string 测量数据返回的weight字符串
返回值
类型 说明
string 对应斤单位数值的字符串,一位有效数字

getWeightStLb

单位转换（kg ==> st:lb）

参数
名称 类型 说明
weight string 测量数据返回的weight字符串
返回值
类型 说明
string [ ] [0] 对应St单位数值的字符串 ，整形 ； [1] 对应Lb单位数值的字符串， 一位有效数字

getWeightSt

单位转换（kg ==> st）

参数
名称 类型 说明
weight string 测量数据返回的weight字符串
返回值
类型 说明
string 对应st单位数值的字符串,两位有效数字

setDeviceUnit

设置设备单位

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
weightUnit QNScaleUnit 重量单位
返回值

类型：number 详情见状态码列表

setMeasureUser

设置测量用户信息

注意: 测量过程中,如果未及时设置测量用户信息。这时收到测量数据结果是不完整的,需要调用补全数据方法才能获取到完整数据。

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
user QNUser 用户信息
返回值

类型：number 详情见状态码列表

setFatMeasurementSwitch

设置秤端是否测脂

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
state boolean 开关状态 true开启测脂 false关闭测脂
longTerm boolean 开关是否长期有效
返回值

类型：number 详情见状态码列表

setBuzzerSwitch

设置秤端蜂鸣器开关

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
state boolean 开关状态 true开启蜂鸣器 false关闭蜂鸣器
返回值

类型：number 详情见状态码列表

startConnectWiFi

开始配置设备WiFi

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
wifiInfo QNScaleWiFiInfo WiFi信息
返回值

scanNearbyWiFi

扫描附近WiFi

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
返回值

类型：number 详情见状态码列表

setMeasureUserToUserDevice

设置用户秤秤端测量用户信息

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
user QNScaleUser 秤端用户信息
返回值

类型：number 详情见状态码列表

deleteUserList

删除秤端用户信息

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
useIndexList number[] 要删除用户编号列表，范围 1~8
返回值

类型：number 详情见状态码列表

updateUserIdentifyWeight

设置秤端用户识别体重

参数
名称 类型 说明
device QNScaleDevice 家用秤设备
weight number 体重 单位kg
返回值

类型：number 详情见状态码列表
