# QNKitchenScaleDataListener

设备数据监听器，用于监听回调设备测量数据；

## API

### onGetRealTimeData

设备实时测量数据回调

##### 参数

| 名称     | 类型                                                | 说明      |
|:-------|:--------------------------------------------------|:--------|
| data   | [QNKitchenScaleData](../Model/QNKitchenScaleData.md)     | 厨房秤测量数据 |
| device | [QNKitchenScaleDevice](../Model/QNKitchenScaleDevice.md) | 厨房秤设备   |
