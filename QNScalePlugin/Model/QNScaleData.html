makeDataComplete

若数据的isDataComplete为false，即数据不完整，调用此方法，传入用户信息以计算完整的测量数据
若数据的isDataComplete为true，则数据已经是完整的，不需要调用此方法。如果调用，则直接返回数据本身
若传入的user为undefined，则不计算直接返回数据本身

参数
名称 类型 描述
user QNUser 用户信息
返回值
名称 类型 描述
scaleData QNScaleData 测量数据
