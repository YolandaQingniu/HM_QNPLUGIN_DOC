QNShareData

QNShareData

共享秤数据对象

成员变量

名称 类型 描述
sn string 共享秤序列号
qnScaleData QNScaleData 测量数据

decodeShareData

解析共享秤二维码

参数
名称 类型 描述
qrcode string 二维码对应的url
validTime number 二维码的有效时间，单位s，为0表示一直有效，不允许传入负值
qnUser QNUser 此数据对应的用户信息
返回值
类型 描述
QNShareData | undefined 共享秤数据对象，解析失败时，返回undefined
