# QNRulerData

围度尺测量数据

#### 成员变量

| 名称      | 类型                                        | 描述                                     |
| --------- |-------------------------------------------|----------------------------------------|
| value     | string                                    | 测量长度，单位为CM时，精确到小数点后1位；单位为IN时，精确到小数点后2位 |
| unit      | [QNRulerUnit](../Constant/QNRulerUnit.md) | 测量长度对应的单位                              |
| timeStamp | string                                    | 时间戳，以秒计算                                |
