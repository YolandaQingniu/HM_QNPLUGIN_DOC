基础鉴权插件

概述

该文档介绍基础鉴权插件的接入，使用各个设备的插件前，必须先初始化基础鉴权插件

对接步骤

1、引入配置文件

将我司商务人员提供的后缀为.qn的配置文件，放到app项目工程的AppScope/resources/rawfile目录下

2、初始化基础鉴权插件

调用initSdk方法，初始化SDK

若初始化失败，错误码详见状态码列表

    //伪代码示例
    const qnPlugin = QNPlugin.getInstance(applicationContext)
    if (undefined != qnPlugin) {
      qnPlugin.initSdk(我司商务提供的appId, 项目工程中配置文件的文件名, {
        onResult: (code: number, msg: string) => {
          AppLogger.logE(`初始化SDK结果 ${code} ${msg}`)
          if (code == 0) {
            //初始化基础鉴权插件成功
          }
        }
      } )
    }

3、初始化成功后，按需接入具体设备插件
